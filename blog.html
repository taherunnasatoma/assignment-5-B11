Question-2: What is the difference between innerHtml,innerText and textContent?

In JavaScript, innerHTML, innerText, and textContent are all properties of DOM elements that allow to retrieve or modify the content of an element.
innerHTML allows you to get or set the HTML content inside an element. It returns the HTML markup within an element, including any child elements, text, or HTML tags

Question-3: What is event delegation in the DOM?

Ans:Event delegation is a technique in JavaScript where you attach a single event listener to a parent element rather than adding individual event listeners to each child element.

Question-4: What is event bubbling in the DOM?

Ans:Event bubbling is a technique in JavaScript where an event happens on an element ,it first runs the handelers on it then on its parent ,then all the way up on other ancesstors.
